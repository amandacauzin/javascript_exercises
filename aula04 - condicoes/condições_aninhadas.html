Condições AninhadasCONDIÇÕES ANINHADAS -> condições dentro de condições Exemplo: if (cond1){ bloco1 } else{ if(cond2){ bloco2 } else{ bloco3 } } CONDIÇÕES MULTIPLAS -> varias condições que finalizam com um mesmo resultado -> é necessário inserir o BREAK -> switch testa valores pontuais, numeros inteiros e caracteres, não intervalos como no if switch(expressão){ case valor 1: bloco 1 break case valor 2: bloco 2 break case valor 3: bloco 3 break default: bloco caso não seja seguido nenhum dos casos break }
